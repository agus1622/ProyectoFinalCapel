Esta aplicación web simula transacciones básicas de un homebanking: login, transferencias, compra de dólares, plazo fijo y consulta de movimientos.

##  Descripción

Las transferencias entre usuarios impactan en los saldos y movimientos tanto de la cuenta de origen como de destino.  

Los datos se almacenan utilizando **localStorage**, permitiendo persistencia en el navegador.  

Se realizan validaciones de:
- usuario y PIN  
- disponibilidad de fondos  
- campos completos
